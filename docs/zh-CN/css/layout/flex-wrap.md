# flexWrap

`flexWrap` 定义了子元素如何在接触到父容器底部时执行换行的行为。默认值为 `nowrap`。

| Name               | Description      | Type                         | Default |
|--------------------|------------------|------------------------------| ------- |
| flexWrap         |[MDN 文档](http://developer.mozilla.org/zh-CN/docs/Web/CSS/flex-wrap) | ^[enum]`'wrap' \| 'nowrap'`| `nowrap`|
