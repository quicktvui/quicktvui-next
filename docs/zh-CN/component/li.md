# li

## API

### Attributes

| Name                 | Description        | Type                                       | Default |
| -------------------- | -------------------| -------------------------------------------|---------|
| type            | 指定一个函数，在其中返回对应条目的类型（返回Number类型的自然数，默认是0），List 将对同类型条目进行复用，所以合理的类型拆分，可以很好地提升list 性能。 | `number`| 0|       |
| key             | 指定一个函数，在其中返回对应条目的 Key 值 | `string`                                    |-|
