# 配置调试模式

## 一、配置调试焦点

* 打开焦点调试

 ``` shell
qui debug -f true
 ```

<img src="/guide/focus/debug-focus.png" />

* 关闭焦点调试

 ``` shell
qui debug -f false
 ```

## 二、配置调试服务地址

 ``` shell
qui debug -s 192.168.80.158
 ```

<img src="/tool/cli/cli_debug_server_host.png" />
