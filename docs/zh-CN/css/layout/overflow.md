# overflow

`overflow` 定义了子元素超过父容器宽高度后的显示情况 `overflow: hidden` 的情况会导致子元素被父容器切割超出显示范围的部分 `overflow: visible` 会让子容器正常显示全部，即使超出父容器的显示范围。

| Name               | Description      | Type                         | Default |
|--------------------|------------------|------------------------------| ------- |
| overflow         |- | ^[enum]`'visible' \| 'hidden'` | `hidden` |
