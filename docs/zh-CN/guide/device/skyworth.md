# 创维电视

## 打开 USB 调试教程

### 通用方法一：

***第一步、找到【本机信息】。***

***第二步、在【本机信息】的界面，用遥控器顺序缓缓输入【上】【上】【下】【下】【左】【右】【左】【右】即可打开工广菜单。***

### 通用方法二：

***第一步、首先需要进入创维电视的【设置】界面，找到【关于电视】。 在其中找到【软件版本号】并连续点击7次即可开启【开发者选项】***。

***第二步、然后在【开发者选项】中找到【USB 调试】选项，将其开启。***

::: warning 注意：打开 ADB 开关之后，部分设备可能需要断电重启方可生效。
:::
