# 海信电视

## 打开 USB 调试教程

### 通用方法一

***第一步、用遥控选择【菜单】，进入【声音设置】里，选择【平衡】。***

***第二步、输入 0538，即可进入工厂模式。***

### 通用方法二

***第一步、打开电视到主界面，按遥控器上的【菜单】键或 【Home】键，进入主菜单。***

***第二步、选择【设置】。在【设置】中找到【关于】选项。***

***第三步、进入【关于】页面后，按遥控器上的【数字7】 或者 【确定】键七次，会出现【开发者模式】选项。***

***第四步、进入开发者模式，找到【USB调试】选项，将其开启。***
