# Android 调试桥 (adb)

[点击查看ADB使用指南](https://developer.android.google.cn/tools/adb?hl=zh-cn)

## 连接设备

 ``` shell
qui adb connect 192.168.90.90
 ```

## 断开指定设备

 ``` shell
qui adb disconnect 192.168.90.90
 ```

## 断开所有设备

 ``` shell
qui adb disconnect
 ```

## 显示所有连接设备

 ``` shell
qui adb devices
 ```
