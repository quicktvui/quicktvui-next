# textAlign

`textAlign` 指定行内文字如何相对它的块父元素对齐，默认值为 `left`。

| Name               | Description      | Type                         | Default |
|--------------------|------------------|------------------------------| ------- |
| textAlign         |[MDN 文档](http://developer.mozilla.org/zh-CN/docs/Web/CSS/text-align) | ^[enum]`'left' \| 'center' \| 'right'`| `left`|
