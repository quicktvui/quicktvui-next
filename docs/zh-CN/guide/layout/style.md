## 样式主要依靠 css、sass、less 样式来实现。

```vue

<div class='div' />
```

```css
.div {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

## 只支持部分CSS样式

::: warning 注意：

***1、不支持的 CSS 样式如下：***
