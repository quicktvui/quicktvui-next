# 播放控制

* 开始播放：`start`
* 暂停播放：`pause`
* 恢复播放：`resume`
* 停止播放：`stop`
* 播放下一个源：`playNextMediaSource`
* 播放上一个源：`playPreviousMediaSource`
* 根据id播放源：`playMediaSourceById`
* 根据index播放源：`playMediaSourceByIndex`

::: warning 注意：
[`start`和`resume`区别、`pause`和`stop`区别](/zh-CN/component/player/page-player#一、播放常用方法简介)
:::
