# display

QuickTVUI 默认采用 Flex 布局。同时，因为仅支持 Flex 布局，所以不需要手写 `display: flex` 即可使用。

| Name               | Description      | Type                         | Default |
|--------------------|------------------|------------------------------| ------- |
| display         |- | ^[enum]`'flex'`| `flex`|
